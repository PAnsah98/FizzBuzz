The Fizz Buzz

This is a Fizz Buzz game, that prints each number from 1 to 100, for each multiple of the first value, it prints "FIZZ" instead of the number. For each multiple of the second value, it prints "BUZZ" instead of the number. For numbers which are multiples of both the first and second values, it prints "FizzBuzz" instead of the number. Ready??
